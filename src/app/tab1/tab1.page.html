Tutorial

Nessa página você poderá encontrar um guia de como criar um aplicativo para celular

Ferramentas:

Para criar um aplicativo mobile iremos usar o IONIC, NodeJS e o Angular, por isso tenha eles instalados no seu computador.

Além disso, iremos programar usando o Visual Studio Code

Criação:

Para criar o aplicativo é necessário abrir o CMD e ir até a pasta "C:\ionic", uma vez que estamos dentro dessa pasta podemos criar nosso APP em etapas ou em uma só linha.

Etapas:

  • O primeiro comando é o "ionic start";
  • Depois disso iremos escolher o framework Angular (basta selecionar com a setinha do teclado e dar enrter);
  • Após isso, iremos dar um nome ao nosso aplicativo (é comum usar o seguinte modelo: exemploApp);
  • Feito isso iremos decidir qual template iremos ter como base no nosso aplicativo

Templates:

O Ionic nos fornece diversos modelos base de aplicativos.

A gente pode destacar o Blank, que é um modelo que simples, sem nada.

Temos também o TABS, que foi usado nesse App e nele podemos ter essa barra de navegação no final da tela.

Além desses, podemos citar também o SideMenu, que, pasmem, nos da um menu lateral onde podemos navegar entre as páginas.

Existem outros modelos e todos eles são resumidos (em inglês) na hora de escolher no CMD.

Em uma linha

Podemos pupar todo o esforço das etapas escrevendo uma simples linha no CMD:

ionic start nomeApp blank --type=angular

Conclusão

PRONTO!! Você criou um app, agr basta usar o comando "cd /nomeApp" e depois "code ." para abri-lo no Visual Code

Ah, claro! Assim que você criar um app e prograna-lo você irá querer ver como ele está ficando, para iso use o comando "ionic serve" no CMD (Poderá usar "ionic serve -l" para uma visão mais completa, para isso será necessário instalar o Ionic Lab, é só dar YES quando pedir).
